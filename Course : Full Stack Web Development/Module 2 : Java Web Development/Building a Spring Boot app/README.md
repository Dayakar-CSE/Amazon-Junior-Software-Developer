Activity: Building a Spring Boot app
You’re building an online store. The admin needs a secure, straightforward way to manage products—adding, updating, and viewing items with ease. Beyond that, they want to monitor the application’s health to ensure it’s always running smoothly.

In this lab, you’ll create a Spring MVC application that provides this functionality. Using Spring Boot Actuator, you’ll set up real-time monitoring to track application performance and health. This project builds on your skills in MVC architecture, form handling, and role-based access control, giving you hands-on experience in building secure, production-ready applications.

Goal

This lab guides you in building a Spring Boot e-commerce application that lets administrators manage products through a secure web interface. By implementing an MVC architecture, adding user roles with role-based access control, and enabling application monitoring with Spring Boot Actuator, you'll gain hands-on experience in developing a secure, production-ready application.

In this lab, you’ll build a product management application with these key components:

MVC architecture: Establish a clear Model-View-Controller structure.

Product management: Allow secure creation, updating, and display of products.

User roles and security: Implement user roles, including USER and ADMIN, with role-based access control.

Forms with Thymeleaf: Design forms to handle product data using Thymeleaf templates.

Spring Boot Actuator: Enable Actuator endpoints to monitor application health and performance.
