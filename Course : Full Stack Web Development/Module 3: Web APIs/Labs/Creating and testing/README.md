Activity: Creating and testing
In any digital app, reliability and accuracy are key to providing a smooth experience and managing data correctly. Think about an online library system where users can check out available books, borrow them, and return them when done. Thorough testing is essential to keep these functions running smoothly.

In this lab, you’ll work with code from a digital library e-commerce app, guiding you through testing its RESTful services. These services include:

Retrieving book details (GET /books/{id})

Borrowing a book (POST /books/{id}/borrow)

Returning a book (POST /books/{id}/return)

Goal

Ensure your e-commerce app runs reliably by testing RESTful services with manual and automated methods, using Postman and Spring Boot’s testing framework for real-world stability.

The goal is to ensure that your e-commerce application performs reliably whether users are browsing available books or checking one out. You'll learn how to test your Spring Boot app using tools like Postman and Spring Boot’s testing framework, helping you catch issues early and build a more resilient application. This experience will prepare you for real-world scenarios where automated and manual testing is essential for application stability and reliability.

By the end of this lab, you’ll be able to:

Manually test RESTful services using Postman.

Write unit tests for controllers to ensure individual methods work as expected.

Create integration tests to check the full flow of requests in your app.

Verify the setup and use of MySQL with your Spring Boot application.
