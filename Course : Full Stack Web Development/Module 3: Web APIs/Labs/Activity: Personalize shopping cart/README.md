As a junior developer at FullStackCart, a fast-growing online marketplace, you’re tasked with building essential features like user registration, login, search functionality, and cart management. These tools are critical to enable users to create accounts, log in, search for products, and manage items in their carts.

In this lab, you'll take on the role of building and testing these core e-commerce features. Through these tasks, you’ll gain valuable experience with frontend JavaScript, HTML, CSS, and RESTful API integration—key skills for a junior software developer.

Goal

Your goal is to develop essential frontend skills with JavaScript, HTML, CSS, and RESTful APIs by building practical e-commerce features for an online store.

🖥️  Note: When you encounter this icon, it’s time to get into your IDE and start coding!

Getting started
Open Visual Studio Code (VSCode) in your lab environment by double-clicking on the icon.


🖥️  Note: Ensure that the folder ajsd_c5m3l2_item_08_activity_personalize_shopping_cart_starter_code is opened as the project root (parent directory) in VS Code.

In this lab, you'll have to complete five tasks in order to perform:

Database setup:

You'll work with the FullStackCart MySQL database (database dump link provided). Run a SELECT query on the products table to confirm the database is set up correctly. The credentials are as follows: username root and password password.

Backend setup:

This lab includes Spring Boot Backend RESTful APIs (link provided). The backend application should be started automatically, you can check localhost:8080 to confirm. If the backend is not started you can open IntelliJ navigate to Application.java and run the main() method to start the backend. Then, explore the controller classes for the API endpoints. You can test these by mocking them in Postman.

API testing:

To verify API endpoints, open Postman and follow the lab steps to check the responses and troubleshoot if necessary.
